# Lexel Employee Status Tracker

A full-stack web application for tracking employees' daily tasks and team progress. Built with React, Express.js, and MongoDB.

## Quick Start

### Prerequisites

- Node.js 18+
- MongoDB (local or Atlas)

### 1. Server Setup

```bash
cd server
npm install
npm run dev
```

### 2. Client Setup

```bash
cd client
npm install
npm run dev
```

### 3. Access

- **Employee App**: http://localhost:5173
- **Admin Panel**: http://localhost:5173/admin/login
- **API**: http://localhost:5000/api

### Default Admin Login

- Email: `admin@lexel.com`
- Password: `Admin@123`

## Tech Stack

- **Frontend**: React 18 + Vite + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: MongoDB + Mongoose
- **Auth**: JWT
